🧠 MEMORY AIDS

Test Lifecycle (easy to remember):
  1️⃣  beforeAll()  ← Set up database (once)
  2️⃣  beforeEach() ← Clean data (before each test)
  3️⃣  TEST CODE    ← Your test logic
  4️⃣  afterEach()  ← Optional cleanup (after each test)
  5️⃣  afterAll()   ← Close database (once)

HTTP Methods (CRUD operations):
  POST   → CREATE (201 expected)
  GET    → READ   (200 expected)
  PUT    → UPDATE (200 expected)
  DELETE → DELETE (200 expected)

Status Codes (remember 3 numbers):
  2XX → Success ✅
  4XX → Client error ❌
  5XX → Server error 💥
